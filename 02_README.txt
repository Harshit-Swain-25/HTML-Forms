Contact Form Project (HTML Only)

This is a simple contact/registration form built using only HTML.

How to use:
1. Open the file contact-form.html in your browser.
2. Fill the form and click Submit to see the data in the browser URL.
3. Edit the form in VS Code to add or remove fields.

Features:
- Text, email, number, and phone inputs
- Radio buttons and checkboxes
- Dropdown menu, date picker, and textarea
- Submit and Reset buttons
- Works without any CSS or JavaScript
