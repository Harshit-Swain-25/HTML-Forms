# Contact / Registration Form

A simple Contact / Registration Form built using pure HTML. This project demonstrates various HTML form elements, input validations, and user interaction components without requiring a backend server.

## 📌 Project Overview

This project showcases fundamental HTML form concepts commonly used in web development.

Users can:

- Enter personal information
- Choose a preferred session
- Select multiple interests
- Choose their experience level
- Pick a preferred start date
- Write additional notes
- Adjust an excitement level slider
- Choose a favorite color

The project uses the GET method to demonstrate how form data can be passed through the URL.

---

## 🚀 Features

- Simple and user-friendly interface

- Multiple checkbox selection

## 📝 How It Works

The form uses:

```html
method="get"
action="#"
```

When the form is submitted, the page reloads and appends the entered values to the URL.

Example:

```text
?name=Harshit&email=you@example.com&session=morning&level=beginner
```

This project is intended for learning and practicing HTML form creation.
